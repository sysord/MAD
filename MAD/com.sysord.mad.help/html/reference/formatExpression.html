Format Expression

Format Expressions are used for all the labels definable in the MAD configuration.

A Format Expression is a string expression which can contain queries that will be evaluated for the context element.
To define a query inside a format expression, the query must be put between [ ].

Example : Format expression of the UML NamedElement label provider 

	Configuration NamedElement for uml.NamedElement {
	    // Will provide the string "Element : " with the name of the NamedElement
	    label provider: "Element : [name]"
	
	    template:
	    // [...]
	}
