Imports

To use EMF metamodels or other MAD configurations, the configuration must import its dependencies.
The import instruction is done with the keyword import at the beginning of the file and a string representing the URI of the file to import.

Metamodels

An EMF metamodel can be imported with an URI of its ecore file.

Example : Import of the UML metamodel 

	import "platform:/plugin/org.eclipse.uml2.uml/model/UML.ecore"

MAD Configuration files

The MAD configuration can also import other MAD configuration files.

Example : Import of the provided config.mad 

	import "platform:/plugin/com.sysord.mad.core/model/config.mad"
