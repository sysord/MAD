UI Tabs

Widgets can be devided up into several defined tabs on a view.

Declaration

A Tab is defined with an ID and a label that will be used for the tab name. 

	UI Tab {
	    id: TAB_ID
	    label: "Tab Name"
	}

Usage

To put a widget in a tab, the widget must specify the ID of the tab.

Example : Text widget that will be put in the tab TAB_ID 

	widget: Name
	tab: TAB_ID // ID of the tab into which put this widget
	label: "Name"
	type: TEXT_WIDGET
	value: "name"
